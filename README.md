   ```sh

   cd ai-code-reviewer
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and configure:
   ```env
   GOOGLE_GEMINI_API_KEY=your_google_gemini_api_key
   PORT=3000
   ```

4. **Start the backend:**
   ```sh
   cd Backend
   npx nodemon
   ```

5. **Start the frontend (React):**
   ```sh
   cd Frontend
   npm run dev
   ```
